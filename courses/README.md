# 📚 Courses

## Description
This folder contains all course materials related to GIS, GeoAI, Remote Sensing, and Spatial Data Science.

## Content
- Lecture notes
- Tutorials
- Step-by-step guides
- Concepts explanations

## Topics Covered
- Introduction to GIS
- Spatial Data Models (Vector & Raster)
- Coordinate Systems & Projections
- Remote Sensing Fundamentals
- GeoAI Concepts
- Urban Planning Applications

## Objective
Provide structured and progressive learning materials to help students build strong foundations in geospatial technologies.

## How to Use
1. Start with introductory lessons
2. Follow the order of topics
3. Combine with exercises in the `exercises/` folder
