# SIG Fundamentals

## Description
Ce dépôt contient des ressources d’introduction aux Systèmes d’Information Géographique pour les étudiants et les débutants.

## Contenu
- Cours de base
- Exercices pratiques
- Jeux de données
- Illustrations pédagogiques

## Objectifs pédagogiques
- Comprendre les concepts fondamentaux du SIG
- Manipuler des données géographiques
- Réaliser des analyses simples
- Produire des cartes

## Structure
- `courses/` : cours et supports
- `exercises/` : travaux pratiques
- `datasets/` : données d'exemple
- `images/` : captures et illustrations

## Public cible
Étudiants, débutants et professionnels souhaitant renforcer leurs bases en SIG.
