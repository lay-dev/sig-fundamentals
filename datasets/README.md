# 📊 Datasets

## Description
This folder contains datasets used for exercises, projects, and demonstrations.

## Types of Data
- Vector data (Shapefiles, GeoJSON)
- Raster data (Satellite imagery, DEM)
- Tabular data (CSV, Excel)

## Examples
- Administrative boundaries
- Land use maps
- Satellite images (Sentinel, Landsat)
- Urban planning data

## Objective
Provide real-world data for practical learning and experimentation.

## Usage Guidelines
- Use datasets for educational purposes
- Combine datasets with exercises and projects
- Always understand the data before analysis

## Note
Some datasets may come from open data sources or public repositories.
